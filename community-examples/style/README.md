# Community Style Examples

This directory contains style.md examples contributed by the ship community. Use these as inspiration for crafting your own unique voice!

## Why Share Your Style?

- Help others learn how to customize their voice
- Showcase different writing styles and use cases
- Build a library of real-world examples
- Get feedback from the community

## How to Contribute

1. **Create your style file**: Copy your customized `style.md` to this directory
   - Name it: `your-name.md` or `your-handle.md`
   - Example: `john-doe.md` or `@username.md`

2. **Add context** (optional): Include a brief comment at the top explaining:
   - Your target audience (e.g., "Tech entrepreneurs", "JavaScript devs")
   - Your niche/focus (e.g., "AI/ML", "Web3", "DevOps")
   - What makes your style unique

3. **Submit a PR**:
   ```bash
   git checkout -b add-style-example-yourname
   git add community-examples/style/your-name.md
   git commit -m "Add style example: Your Name"
   git push origin add-style-example-yourname
   ```

4. **Open a Pull Request** on GitHub

## Guidelines

- ✅ Real, working style files you actually use
- ✅ Remove any private/sensitive information
- ✅ Keep brand-specific details generic or clearly labeled
- ❌ No generic/template examples - we want YOUR voice
- ❌ No spam or promotional content

## Example Contributions

Browse the files in this directory to see different approaches:
- Casual vs. professional tone
- Technical vs. business-focused
- Short punchy vs. detailed storytelling
- Different emoji/hashtag strategies
- Various target audiences
